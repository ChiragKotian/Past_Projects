3D Printed RC Plane

In this project, we made an RC plane with the majority of parts 3D printed (ABS material).

Introduction

Cutting Foam and modelling plane is a very delicate and time consuming process. What if we just 3D cad model parts and print it and make the plane out of these parts? Is this feasible? Will the plane fly?

In this project, we explored these questions and made an RC plane with the majority of parts 3D printed (ABS material). The plane didn't fly but we got some valuable insights. We did this project to get hands on experience with the world of 3D printing and RC planes.
