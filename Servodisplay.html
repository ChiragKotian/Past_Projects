Servo Display

This project was a demonstration project we made for Arduino workshop conducted for our batchmates through the robotics club of IIT Tirupati, Techmaniacs.

Introduction

In this project we used arduino to control multiple servos. Each servo controls one segment from the seven segment display and a IR proximity sensor triggers the count. This project can be used as person counter at events.
